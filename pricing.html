Device Service Level Details

Device Type Write Transaction Limits API Writes per minute Read Transaction Limits API Reads per day Socket Service
Low Transaction Volume 0.1 per minute 1 write every 10 minutes 500 No Sockets
Average Transaction Volume 1 per minute 1,000 Includes Sockets
High Transaction Volume 10 per minute 5,000 Includes Sockets
Details
  • Write transactions are metered as transactions per minute measured as a 5 minute moving average
  • Read transactions are metered as transactions per day, counter resets at midnight UTC
  • Device rate limits apply to a device and to all channels on that device
  • Socket listening does not affect read limits
  • HTTP methods over sockets are treated like standard API calls for metering
  • Data is archived for as long as you are paying for device service
  • Channel pricing is per channel per year
